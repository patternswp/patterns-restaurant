# Patterns Restaurant

> **Patterns Restaurant** is a stylish and modern block-based WordPress theme designed for restaurants, cafés, bistros, and food service businesses.

## Description

Patterns Restaurant is a stylish and functional Full Site Editing (FSE) WordPress theme, perfect for creating a mouth-watering online presence for restaurants, cafés, bistros, and food delivery services. Designed to showcase menus, signature dishes, and customer reviews, this block-based theme empowers you to create a unique and visually appealing website. With its FSE capabilities, Patterns Restaurant enables seamless customization of headers, footers, layouts, and global styles directly within the WordPress Site Editor.

## Features

- **Full Site Editing (FSE):** Customize every aspect of your website using the WordPress block editor.
- **Elegant Design:** Perfect for restaurants, cafés, bistros, and food delivery services.
- **Responsive Layouts:** Optimized for seamless display on all devices.
- **Pre-designed Templates:** Includes layouts for showcasing feature sections, services, about, appointments, prices, portfolios, team, testimonials, contact page, and more.
- **Custom Block Patterns:** Simplify website creation with ready-to-use patterns.
- **SEO Optimized:** Built with best practices for improved search engine rankings.
- **High Performance:** Lightweight and optimized for fast loading.

## Getting Started

### Installation

- Clone the repository to your `/wp-content/themes/` directory:

  ```bash
  git clone https://github.com/patternswp/patterns-restaurant.git
  ```

- Or download and upload the theme files to `/wp-content/themes/patterns-restaurant`.

### Activate the Theme

Activate the theme via the WordPress admin dashboard under **Appearance > Themes**.

### Start Customization

With Full Site Editing support, you can customize the theme through the WordPress Site Editor by navigating to **Appearance > Editor**. Modify headers, footers, layouts, and global styles to suit your restaurant needs.
